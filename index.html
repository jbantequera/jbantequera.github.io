Component Host — c-hello-world

Setup Checklist

  1. Create the app in Salesforce:
    Setup → Apps → Lightning Out 2.0 App Manager → New
    Add c-hello-world as a component and save.
  2. Copy the generated script tag from the App Manager and replace the <script src="PASTE_..."> at the top of this file with it.
  3. Copy the 18-digit App ID and replace REPLACE_WITH_18_DIGIT_APP_ID on the <lightning-out-application> element.
  4. Add CORS: Setup → CORS → add http://localhost:8000
  5. Authenticate & get the frontdoor URL via the UI Bridge API:
    Exchange your OAuth access token for a frontdoor_url, then set it on the frontdoor-url attribute (or do so dynamically with JS at runtime).
  6. Serve locally — file:// won't work, use:
    python3 -m http.server 8000 then open http://localhost:8000/lightning-out-2.html
💡 Component naming: LWC helloWorld becomes c-hello-world in Lightning Out 2.0 (camelCase → kebab-case with c- prefix).
